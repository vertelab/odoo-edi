Description

This integration provides information that can form the basis for decisions on support. An endpoint return an enum value of the jobseeker's KROM status (Enum values: Krom, KromEsf, EjKrom). The API is an HTTP proxy against the underlying API of AIS. It is used to request information needed as a basis for decisions on support. More detailed flow documentation is available at the IPF API Portal: AIS BOS Regelverk, endpoint /kromtyp-for-postnummer.

Configuration

To configure this integration, you need to:

  1. Go to Tekniskt > EDI > EDI Route
  2. Edit or create AF Asok: Get AIS BOS route
  3. Go toProtocol settings menu and enter the following parameters
    • Url: https://ipf-acc.arbetsformedlingen.se
    • Port: 443
    • Client ID: is available in the IPF Portal
    • Client Secret: is available in the IPF Portal
    • SSL-Protocol: Simple verification
    • AF-Environment: T2
    • AF-Systemid: AFCRM

Test

To test this integration, you need to:

  1. Go to Tekniskt > Åtgärder >Serveråtgärder
  2. Edit or create EDI AF Get AIS BOS
  3. The customer_id should be set to a jobseeker that has a valid postcode. i.e 721 34
  4. Return to Jobseeker tab and confirm the field Krom is set.
